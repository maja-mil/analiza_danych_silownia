## Opis projektu
Projekt polega na analizie danych dotyczących aktywności użytkowników siłowni (cechy demograficzne, parametry fizjologiczne oraz wskaźniki treningowe). Celem jest sprawdzenie zależności między zmiennymi (m.in. czas trwania treningu, spalone kalorie, tętno, doświadczenie, nawodnienie) oraz wykonanie podstawowego wnioskowania statystycznego i segmentacji użytkowników.

## Zawartość folderu
- Raport.Rmd – główny raport (opis + analizy + wykresy)
- dane/silownia_new.csv – dane wejściowe do analizy

## Jak uruchomić
1. Otwórz plik Raport.Rmd w RStudio.
2. Upewnij się, że plik danych znajduje się w folderze dane/ (ścieżka: dane/silownia_new.csv).
3. Jeśli brakuje pakietów, zainstaluj je.
